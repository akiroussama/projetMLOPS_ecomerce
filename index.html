Phase 1

Conteneurisation Docker

Dockerfile API
FROM python:3.10-slim
WORKDIR /app

# Dependencies
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Artefacts montes en volume
RUN mkdir -p /app/models

# Code source
COPY src/ /app/src/

EXPOSE 8000
CMD ["uvicorn", "src.api.app:app",
     "--host", "0.0.0.0", "--port", "8000"]
Image legere

python:3.10-slim pour minimiser la taille. Les artefacts modele sont montes en volume Docker, pas inclus dans l'image.

Dockerfiles specialises
Service Base Image Role
Dockerfile python:3.10-slim API FastAPI
docker/trainer/ python:3.10-slim Training + Bootstrap
docker/mlflow/ python:3.10-slim MLflow Server
docker/airflow/ apache/airflow Orchestrateur
docker/streamlit/ python:3.10-slim Interface UI
Bootstrap automatique

Le service bootstrap telecharge les donnees S3, pretraite, et entraine le modele initial automatiquement au premier demarrage.
